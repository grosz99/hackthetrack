## Key Interaction Patterns Summary

### SCOUT LANDING PAGE
- **Primary Action**: Click card β†’ Navigate to detail
- **Secondary Action**: Click "+ Compare" β†’ Add to queue
- **Filters**: Apply instantly (no submit button), results update in real-time
- **Search**: Type-ahead with debounce (300ms), shows top 5 matches
- **Sorting**: Dropdown in top bar, applies immediately
- **View Toggle**: Grid β†” Table (icon button in top bar)

### DRIVER DETAIL PAGE
- **Back Navigation**: "β† Scout Portal" button (top-left) or browser back
- **Driver Switching**: Dropdown selector (top-right) β†’ instant navigation
- **Tab Navigation**: Click tab β†’ URL updates, content swaps
- **Comparison**: "Compare with Others" β†’ Opens modal to add to queue
- **Actions**: Share, Export, Add to List (in "More" menu)

### COMPARISON VIEW
- **Add/Remove**: Drag drivers in/out of queue (desktop), swipe (mobile)
- **View Details**: Click driver column β†’ Navigate to detail
- **Export**: Generate PDF or CSV report
- **Return**: "β† Scout Portal" or "β† Back to Grid"

### KEYBOARD SHORTCUTS
- `Alt + S`: Focus search
- `Alt + F`: Open filters
- `Alt + C`: Open comparison queue
- `Alt + β†`: Back to scout (from detail page)
- `Esc`: Close modals/drawers
